# Module 4: 팀 챌린지 - 우리 점포 도우미 만들기 🏆

## 🔥 드디어 본 게임입니다!

Module 1~3에서 배운 모든 것을 총동원해서, **우리 매장에 진짜 필요한 앱**을 만들어봅시다!

| 순서 | 내용 | 시간 |
| --- | --- | --- |
| 1 | 🤝 조 편성 + 주제 선택 | 5분 |
| 2 | 💻 자유 바이브 코딩 | 20분 |
| 3 | 🎤 결과물 발표 + 투표 | 5분 |

> **ℹ️ 이것만 기억하세요!**
> 완벽한 앱을 만드는 게 아닙니다.\
> **"이런 게 있으면 우리 매장에서 진짜 쓰겠다!"** 하는 아이디어를 **눈에 보이게 만드는 것**이 핵심입니다! 💡

***

## 🤝 조 편성 — 최강의 팀을 만들어보세요!

**2인 1조**로 진행합니다.

### 팀 구성 꿀팁 🍯

| 조합 | 장점 |
| --- | --- |
| 🖥️ 같은 지역 동료끼리 | 서로의 매장 상황을 잘 알아서 아이디어가 구체적 |
| 🆕 처음 만난 사람끼리 | 다른 지역의 노하우를 배울 수 있음 |
| 💪 자신감 있는 사람 + 아이디어 많은 사람 | 한 명이 운전, 한 명이 내비게이션! |

### 역할 분담 추천 🎭

| 역할 | 하는 일 |
| --- | --- |
| **🧑‍✈️ 조종사 (Driver)** | 키보드를 잡고 Kiro에 직접 입력하는 사람 |
| **🧭 내비게이터 (Navigator)** | 옆에서 아이디어를 내고 화면을 확인하는 사람 |

> **⚠️ 잠깐!**
> 두 사람 다 각자 화면을 보고 있으면 시간이 두 배로 걸립니다! ⏰\
> **한 화면을 같이 보면서** 하는 게 훨씬 빠릅니다. 노트북 하나에 모여주세요!
