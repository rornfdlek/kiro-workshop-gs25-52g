# 첫 번째 프롬프트

## 앱 만들기, 시작합니다!

우측의 **Kiro Chat** 패널을 찾아주세요.\
이제 이 채팅창에 한글로 요청을 입력하면 됩니다.

***

## Step 1: 프롬프트 입력하기

아래 내용을 Kiro Chat에 복사해서 붙여넣고 Enter를 누릅니다.

**📋 복사해서 붙여넣기**

```
편의점 점주를 위한 규정 검색 도우미 웹페이지를 만들어줘.

- 상단에 "GS25 규정 도우미"라는 제목
- 가운데에 채팅형 질의응답 영역
- 하단에 질문 입력창과 전송 버튼
- 입력창 위에 자주 묻는 질문 예시 버튼 3개: "반품 절차", "유통기한 관리", "야간 근무 규정"
- 예시 버튼을 클릭하면 해당 질문이 자동으로 입력되게 해줘
```

### 프롬프트 각 부분이 하는 일

| 문장 | 역할 |
| --- | --- |
| "편의점 점주를 위한 규정 검색 도우미 웹페이지를 만들어줘" | **무엇**을 만들지 알려줌 |
| "상단에 GS25 규정 도우미라는 제목" | **화면 구성**을 지정 |
| "채팅형 질의응답 영역" | **핵심 기능**을 설명 |
| "자주 묻는 질문 예시 버튼 3개" | **사용 편의성** 추가 |
| "클릭하면 자동으로 입력되게" | **상호작용** 정의 |

> **ℹ️ 참고**
> 프롬프트는 **구체적일수록 좋습니다.**\
> "웹페이지 만들어줘"보다 "채팅형 질의응답 영역이 있는 웹페이지 만들어줘"가 훨씬 좋은 결과를 냅니다.

***

## Step 2: AI가 코드를 생성하는 과정 지켜보기

Enter를 누르면 Kiro가 코드를 만들기 시작합니다.

이때 화면에서 일어나는 일:

1. **채팅창**: AI가 "어떻게 만들겠다"고 설명합니다
2. **코드 영역**: 코드가 자동으로 작성되는 것이 보입니다
3. **파일 탐색기**: 새로운 파일이 생깁니다

> **⚠️ 주의**
> 코드가 만들어지는 동안 **1~2분 정도** 걸릴 수 있습니다.\
> 코드 영역에서 글자가 빠르게 쓰여지는 것이 보이면 정상입니다.\
> 완료될 때까지 기다려주세요!

***

## Step 3: 결과 확인하기

AI가 코드 생성을 완료하면, 만들어진 앱을 브라우저에서 확인합니다.

1. 파일 탐색기에서 `index.html` 파일을 찾습니다
2. 마우스 오른쪽 클릭 > **Open with Live Server** 를 선택합니다
   * Live Server는 만든 앱을 브라우저에서 바로 미리보기 해주는 기능입니다 (미리 설치되어 있습니다)
3. 브라우저에 앱이 열립니다!

> **✅ 완료**
> 브라우저에 채팅 화면이 보이시나요?\
> **축하합니다!** 방금 코드를 한 줄도 쓰지 않고 웹앱을 만들었습니다!

***

## 잘 안 되는 경우

<details>
<summary>브라우저에 아무것도 안 보여요</summary>

* `index.html` 파일이 생성되었는지 파일 탐색기에서 확인하세요
* Live Server가 설치되어 있는지 확인하세요 (진행자에게 문의)
* 브라우저 주소창에 `http://localhost:5500` 또는 유사한 주소가 보이는지 확인하세요

</details>

<details>
<summary>코드 생성이 중간에 멈췄어요</summary>

* Kiro Chat에서 "계속해줘" 또는 "이어서 만들어줘"를 입력해보세요
* 그래도 안 되면 진행자에게 문의하세요

</details>

<details>
<summary>Steering에 적은 디자인이 적용 안 된 것 같아요</summary>

* Steering 파일이 저장되었는지 확인하세요 (Ctrl+S / Cmd+S)
* 채팅에서 "steering에 적은 디자인 규칙을 반영해줘"라고 한번 더 요청해보세요

</details>
