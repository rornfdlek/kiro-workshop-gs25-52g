# 워크샵 소개

## 왜 이 워크샵을 하나요?

여러분은 이미 MISO를 통해 AI 에이전트 워크플로우를 만들어본 경험이 있습니다.\
드래그앤드롭으로 노드를 연결하고, 프롬프트를 작성해서 AI가 일하게 만들어보셨죠.

그런데 한 가지 아쉬운 점이 있었을 겁니다.

> "만든 워크플로우를 **점주나 OFC가 바로 쓸 수 있는 앱**으로 만들 수는 없을까?"

오늘 배울 **Kiro**가 바로 그 답입니다.

***

## 오늘의 목표

> **✅ 완료**
> **코드를 한 줄도 직접 작성하지 않고**, 대화만으로 편의점 업무 도우미 웹앱을 완성합니다.

구체적으로 이런 것들을 체험합니다:

| 체험 항목 | 설명 |
| --- | --- |
| **Steering** | AI에게 프로젝트 규칙을 알려주는 방법 |
| **바이브 코딩** | 자연어 대화로 웹앱을 만들고 수정하는 방법 |
| **Spec** | 요구사항을 정리하면 AI가 설계도를 그려주는 기능 |
| **@파일 컨텍스트** | 내가 가진 데이터를 AI에 연결하는 방법 |

***

## 워크샵 진행 방식

* **2인 1조**로 진행합니다
  * 한 명이 프롬프트를 작성하고, 한 명이 결과를 확인하며 아이디어를 제안합니다
  * 역할을 번갈아가며 진행해주세요
* 모든 작업은 **한글**로 진행합니다
* 막히면 언제든 손을 들어주세요

***

## 오늘 만들 앱 미리보기

워크샵 진행자가 완성된 데모 앱을 보여드립니다.

오늘 워크샵이 끝나면, 여러분도 이런 앱을 직접 만들 수 있게 됩니다:

* 질문을 입력하면 규정을 찾아 답변해주는 **채팅형 웹 페이지**
* GS25 브랜드 컬러가 적용된 **깔끔한 디자인**
* **모바일**에서도 잘 보이는 반응형 레이아웃

> **ℹ️ 참고**
> 데모를 본 후 "저게 정말 코드 없이 만들어지나요?"라는 의문이 드실 겁니다.\
> 네, 됩니다. 직접 해보시죠!
