# GS25 편의점 운영 규정 FAQ (샘플 데이터)

> 이 문서는 워크샵용 샘플 데이터입니다. 실제 GS25 규정과 무관합니다.

---

## 반품/교환

### 규정 #01 - 일반 상품 반품
- **적용 대상**: 유통기한 내 일반 상품
- **처리 기한**: 입고 후 72시간 이내
- **절차**: POS 반품 처리 → 반품 사유 입력 → 물류 회수 요청
- **담당**: 점포 관리자 또는 대행 OFC

### 규정 #02 - 유통기한 임박 상품 처리
- **기준**: 유통기한 D-3일부터 할인 판매 가능
- **할인율**: 최대 30%까지 점장 재량
- **폐기**: 유통기한 당일 22시까지 POS 폐기 처리 필수
- **담당**: 점포 관리자

### 규정 #03 - 불량품 교환
- **처리**: 발견 즉시 사진 촬영 → 본사 품질관리팀 보고
- **연락처**: 내선 2345
- **보상**: 고객 불량품 수령 시 동일 상품 교환 + 사과 쿠폰 제공

## 재고 관리

### 규정 #04 - 정기 재고 실사
- **주기**: 매월 1일, 15일
- **방법**: PDA 스캔 방식
- **보고**: 실사 완료 후 24시간 내 본사 시스템 등록
- **담당**: 점장

### 규정 #05 - 긴급 발주
- **조건**: 재고가 안전 재고 이하로 떨어진 경우
- **방법**: 모바일 발주 앱 또는 POS 긴급 발주 기능
- **처리 시간**: 발주 후 12~24시간 내 입고
- **담당**: 점포 관리자

### 규정 #06 - 신상품 입고
- **알림**: 본사에서 입고 3일 전 통보
- **진열**: 입고 당일 매대 진열 완료 (진열 가이드 참고)
- **POP**: 신상품 POP 부착 필수
- **담당**: 점포 관리자

## 근무 관리

### 규정 #07 - 야간 근무 규정
- **야간 시간**: 22:00 ~ 06:00
- **최소 인원**: 1명 (단독 근무 가능, 단 방범 교육 필수)
- **수당**: 기본 시급의 1.5배
- **휴게**: 4시간 근무 시 30분 휴게 보장
- **담당**: 점장 (근무표 작성)

### 규정 #08 - 교대 근무 인수인계
- **시간**: 교대 시점 전후 15분
- **필수 항목**: 금전 확인, 특이사항 전달, 재고 이상 유무
- **기록**: 인수인계 대장 작성 필수
- **담당**: 교대 당사자

### 규정 #09 - 신규 직원 교육
- **필수 교육**: 위생 교육(2시간), POS 교육(3시간), 고객 응대(1시간)
- **완료 기한**: 근무 시작 후 7일 이내
- **인증**: 교육 완료 후 본사 시스템에 등록
- **담당**: 점장 또는 교육 담당 OFC

## 위생/안전

### 규정 #10 - 매장 청소
- **일일 청소**: 영업 시작 전, 점심/저녁 피크 후, 영업 종료 후 (1일 3회)
- **주간 청소**: 진열대 전체 정리, 냉장고 내부 청소
- **월간 청소**: 에어컨 필터, 환기구, 창고 정리
- **기록**: 청소 체크리스트 작성 필수

### 규정 #11 - 식품 안전 온도 관리
- **냉장**: 0~10°C 유지
- **냉동**: -18°C 이하 유지
- **점검**: 1일 2회 온도 기록 (오전 10시, 오후 4시)
- **이상 시**: 즉시 본사 설비팀 연락 (내선 3456)

### 규정 #12 - 화재 대응
- **초기 대응**: 소화기 사용 → 119 신고 → 고객 대피 유도
- **소화기 위치**: 카운터 뒤, 창고 입구 (2개소)
- **점검**: 월 1회 소화기 상태 확인
- **보고**: 화재 발생 시 즉시 본사 안전관리팀 보고 (내선 1234)

## 고객 응대

### 규정 #13 - 고객 클레임 처리
- **1단계**: 경청 및 사과 ("불편을 드려 죄송합니다")
- **2단계**: 상황 파악 및 해결 방안 제시
- **3단계**: 처리 후 만족 확인
- **기록**: 클레임 대장 작성 (일시, 내용, 처리 결과)
- **에스컬레이션**: 해결 불가 시 지역 담당 OFC에게 연결

### 규정 #14 - 미성년자 주류/담배 판매 금지
- **확인**: 의심 시 반드시 신분증 확인
- **거절 멘트**: "죄송합니다, 신분증 확인이 필요합니다"
- **위반 시**: 영업정지 및 과태료 부과 대상
- **담당**: 전 직원 (교육 필수)

### 규정 #15 - 택배 서비스
- **접수 시간**: 영업시간 내 상시
- **보관 기한**: 도착 후 3일 이내 수령
- **미수령**: 3일 초과 시 반송 처리
- **분실**: 즉시 택배사 + 본사 고객지원팀 연락 (내선 4567)

## 시설 관리

### 규정 #16 - 냉장/냉동 설비 고장
- **즉시 조치**: 상품을 다른 냉장고로 이동
- **보고**: 본사 설비팀 연락 (내선 3456)
- **응급 처리**: 얼음/아이스팩으로 임시 보관
- **기록**: 고장 시간, 영향 상품 목록 작성

### 규정 #17 - POS 장애
- **1차 조치**: POS 재부팅
- **2차 조치**: 본사 IT 지원팀 연락 (내선 5678)
- **비상 판매**: POS 장애 시 수기 판매 대장 활용
- **정산**: 복구 후 수기 판매분 일괄 입력

### 규정 #18 - CCTV 관리
- **저장 기간**: 최소 30일
- **점검**: 주 1회 정상 작동 확인
- **열람 요청**: 경찰 요청 시 점장 확인 후 제공 (본사 보안팀 사전 통보)
- **담당**: 점장

### 규정 #19 - 일일 매출 정산
- **시간**: 영업 종료 시 (또는 24:00 자동 마감)
- **확인**: 현금, 카드, 간편결제 각 금액 대조

- **확인 주기**: 매일 1회 이상
- **확인 방법**: 본사 인트라넷 또는 모바일 앱
- **필독 공지**: 24시간 내 읽음 확인 필수
- **담당**: 점장
